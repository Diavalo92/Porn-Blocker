# Unwanted Site Blocker Chrome Extension

A Chrome extension that automatically blocks Porn sites and enforces SafeSearch.

## Features

1. **Blocks Porn sites**: Automatically closes Porn sites
2. **Blocks search keywords**: Closes tabs when searching for terms like "nsfw", "porn", "hentai", etc.
3. **Enforces SafeSearch**: Automatically enables SafeSearch on Google

## How to Install

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" in the top right corner
3. Click "Load unpacked" and select this folder
4. The extension is now active!
